100% open source and it's free.

The solution is based on our open source rock stars: sipML5 and webrtc2sip. webrtc2sip is a rock star already used by 100+ companies all around the world. It's smart, powerful, fully featured and intuitive.
No need to write a single line, all components are open source and ready for the prime: sip proxy, media encoder, smtp client, database connector, account manager...
